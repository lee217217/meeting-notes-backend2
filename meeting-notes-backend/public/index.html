Meeting input

Paste transcript text, choose the meeting type and output language, then generate structured notes through the serverless API route.

Paste your notes and click Generate output.

The frontend calls /.netlify/functions/generate, and the function sends the request to the AI API.

Set PERPLEXITY_API_KEY in Netlify environment variables. Never place the key in browser-side code.

Generated output

The function returns structured JSON so the page can render a summary, action list, follow-up email, and Markdown export.
