Meeting input

Paste transcript text, choose the meeting type and output language, then generate structured notes through the serverless API route.

Paste your notes and click Generate output.

The single-output route uses /.netlify/functions/generate. The workflow route uses /.netlify/functions/multi-agent-run and returns structured artifacts plus trace data.

Set PERPLEXITY_API_KEY in Netlify environment variables. Never place the key in browser-side code.

Generated output

The single-output route returns structured JSON so the page can render a summary, action list, follow-up email, and Markdown export.
